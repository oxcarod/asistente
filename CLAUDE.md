# Asistente Personal

Carpeta central donde se guarda todo lo relacionado con información personal, ideas, pendientes y datos.

## Estructura

```
asistente/
├── CLAUDE.md          ← Este archivo
├── notas/             ← Notas generales
├── ideas/             ← Ideas y borradores
├── pendientes/        ← Tareas y pendientes
└── datos/             ← Datos y referencias personales
```

## Propósito

- **Notas** — apuntes,recordatorios, información útil
- **Ideas** — proyectos en mente, bocetos, exploraciones
- **Pendientes** — cosas por hacer, tareas, compromisos
- **Datos** — información personal, contactos, referencias

## Cómo usarlo

Dime algo como:
- "Agrega una nota sobre X"
- "Tengo una idea: ..."
- "Anota esto como pendiente"
- "_guarda este dato en datos"
- "Muéstrame todas mis notas"

Yo mantengo la estructura y te ayudo a organizarte.
